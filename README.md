## 🚀 Como executar o projeto

 ### para execultar WinForm C#:

 - esteje usando um maquina com windows 
 -  usando o [.NET6](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)
 - para execultar o projeto recomendado a ultilização do[ Visual Studio](https://visualstudio.microsoft.com/pt-br/downloads/)

agora que tem o minimo para rodar o projeto faça o clone  do projeto por terminal ou mesmo pelo visual stdudio 

```
 git clone  https://github.com/oVitorio/LoginCadastro.git
```

### para execultar o DataBase:

- necessario usar o [xampp](https://www.apachefriends.org/pt_br/download.html)
use o `PHPMYADMIN` e importe o ***logincadastro.sql*** da pasta *database*

## 💪 Como contribuir para o projeto

1. Faça um **fork** do projeto.
2. Crie uma nova branch com as suas alterações: `git checkout -b my-feature`
3. Salve as alterações e crie uma mensagem de commit contando o que você fez: `git commit -m "feature: My new feature"`
